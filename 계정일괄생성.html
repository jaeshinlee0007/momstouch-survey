🍔 맘스터치 계정 일괄 생성

Firebase Auth 계정을 엑셀로 한 번에 생성합니다

Step 1. Firebase API Key 확인

Firebase Console → ⚙️ 프로젝트 설정 → 일반 탭 → 하단 "웹 API 키" 확인
이미 프로젝트 API Key가 입력되어 있습니다.

Step 2. 엑셀 파일 업로드

엑셀 형식: A열=이름, B열=역할(sv/tl/gl), C열=팀, D열=이메일, E열=초기비밀번호
이메일: D열에 입력한 이메일 그대로 사용
권한: sv=기본 / tl=sv+담당팀 조회+대시보드 / gl=전체
📊
클릭하거나 파일을 드래그하세요
맘스터치_계정목록.xlsx 사용 권장

Step 3. 미리보기 확인

이미 존재하는 계정은 자동으로 건너뜁니다. 목록 확인 후 생성 버튼을 누르세요.
